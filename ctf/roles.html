Find your specialty

Coordinator

  • Team strategist and communication hub
  • Assigns challenges and monitors team status
  • Understands all categories well enough to route tasks
  • Ensures nothing falls through the cracks

Documentation Lead

  • Maintains notes and shared knowledge base
  • Logs working payloads and failed approaches
  • Writes post-event reports
  • Keeps the team organized and aligned

Reverse Engineer

  • Analyzes binaries, assembly, and compiled code
  • Uses debuggers and decompilers
  • Understands malware behavior and binary exploitation

Exploitation Specialist

  • Memory corruption issues
  • Buffer overflows
  • Stack/heap exploitation
  • Low-level attack development

Web Exploitation Expert

  • Understands HTTP, cookies, sessions, and APIs
  • Performs SQLi, XSS, CSRF, and logic flaws
  • Heavy Burp Suite user

Crypto Analyst

  • Works with RSA, DES, XOR, hashing, modular arithmetic
  • Understands cryptographic failures
  • Specializes in cracking and pattern recognition

Forensic Specialist

  • PCAP analysis (Wireshark)
  • Memory forensics (Volatility)
  • Disk forensics and file carving (Autopsy, SleuthKit)
  • Understands evidence recovery

OSINT Investigator

  • Search mastery and pivoting
  • Metadata extraction
  • Social engineering awareness
  • Creative puzzle-solving

Network Specialist

  • Understands packets, routing, protocols, and covert channels
  • Diagnoses misconfigurations and unusual network behavior
  • Can spot what ‚Äúdoesn‚Äôt belong‚Äù in traffic captures

Programmer

  • Python, Bash, C/C++, PowerShell, Regex
  • Automates tasks and builds tooling
  • Writes scripts to speed up exploitation or analysis
