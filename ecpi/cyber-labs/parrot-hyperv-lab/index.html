Parrot OS on Hyper-V Laboratory

Download, install, and configure Parrot OS in Hyper-V to connect to Hack The Box.

Prepared for Students by Professor Jennifer Gentry
Estimated Time: 90–120 Minutes Platform: Windows with Hyper-V Focus: Parrot OS & Hack The Box
1 Lab Overview

In this laboratory, you will configure a Windows system using Hyper-V to run Parrot OS as a virtual machine and connect that VM to Hack The Box (HTB). This provides a controlled, isolated environment for penetration testing and cybersecurity practice.

Lab Objectives

  • Explain the role of Hyper-V as a type-1 hypervisor in Windows.
  • Download and verify a Parrot OS ISO image from the official site.
  • Create and configure a Parrot OS virtual machine in Hyper-V.
  • Configure an external virtual switch for internet access.
  • Install VPN tooling in Parrot and connect to Hack The Box.

Prerequisites

  • Windows 10/11 Pro, Education, or Enterprise with support for Hyper-V.
  • Hardware virtualization enabled in BIOS/UEFI (Intel VT-x / AMD-V).
  • At least 8 GB RAM and 60 GB of free disk space recommended.
  • An active Hack The Box account.
2 Hyper-V & Virtualization Concepts

What Is Hyper-V?

Hyper-V is Microsoft’s native type-1 hypervisor. It allows you to run multiple operating systems (virtual machines) directly on your hardware, managed by Windows. This lab uses Hyper-V to host Parrot OS.

Why Use Hyper-V for Hack The Box?

  • Integrated with Windows, no third-party hypervisor required.
  • Supports virtual switches for flexible network design.
  • Allows you to isolate security tooling from your daily-use host system.

Enable Hyper-V on Windows

Perform these steps on the Windows host (not in a VM):

  1. Open Control Panel → Programs → Turn Windows features on or off.
  2. Check Hyper-V (and all sub-options).
  3. Click OK and allow Windows to install the Hyper-V components.
  4. Restart your system when prompted.
  5. After reboot, open the Hyper-V Manager from the Start menu.
3 Download Parrot OS

Parrot OS is a Debian-based distribution designed for security, privacy, and development. You may use Parrot Security Edition or Parrot Home Edition, depending on course guidance.

Obtain Parrot OS from the Official Site

  1. From the Windows host, open a web browser.
  2. Navigate to the official Parrot website: https://www.parrotsec.org or https://parrotlinux.org.
  3. Locate the Downloads section.
  4. Select an appropriate ISO (e.g., Parrot Security KDE/Mate 64-bit).
  5. Download the ISO file and note the folder where it is stored (e.g., C:\ISOs\Parrot).

Optional: Verify Integrity

For a production environment, you should verify checksums (SHA256) and signatures provided by the Parrot team. For this introductory lab, we focus on basic installation and configuration.

4 Create the Parrot OS Virtual Machine in Hyper-V

Step 1 – Create an External Virtual Switch

An external virtual switch allows your VM to access the internet via the physical network adapter.

  1. Open Hyper-V Manager.
  2. In the right-hand Actions pane, select Virtual Switch Manager….
  3. Choose New virtual network switch → External → Create Virtual Switch.
  4. Give it a clear name, such as External-Parrot-HTB.
  5. Select the physical network adapter used for internet access.
  6. Leave other settings at defaults unless instructed otherwise and click OK.

Step 2 – Create a New VM

  1. In Hyper-V Manager, right-click your host name and select New → Virtual Machine….
  2. Click Next through the introduction.
  3. Assign a name such as Parrot-HTB and choose a storage location if needed.
  4. Select a Generation 2 VM (if Parrot supports your hardware; otherwise, Generation 1).
  5. Assign at least 4096 MB (4 GB) of memory (more if available).
  6. Choose Use dynamic memory if appropriate for your system.
  7. Connect the VM to the external virtual switch you created (e.g., External-Parrot-HTB).
  8. Create a virtual hard disk (e.g., 40–60 GB, dynamically expanding).
  9. On the “Installation Options” page, select Install an operating system from a bootable image file and browse to the Parrot ISO.
  10. Review the summary and click Finish.

Step 3 – Adjust Secure Boot (If Needed)

Many Linux distributions require a different Secure Boot template or disabled Secure Boot.

  1. Right-click the new VM and choose Settings….
  2. Under Security, uncheck Enable Secure Boot OR select a Linux-compatible template if available.
  3. Click OK.
5 Install Parrot OS in the VM

Now you will perform the Parrot OS installation inside the virtual machine.

  1. In Hyper-V Manager, select the Parrot-HTB VM and click Connect….
  2. In the VM window, click the green Start button.
  3. When the Parrot boot menu appears, choose the appropriate Install or Graphical Install option.
  4. Follow the Parrot installer prompts:
    • Select language, location, and keyboard layout.
    • Partition the virtual disk using the guided option.
    • Create a user account and secure password.
  5. Allow the installer to copy files and configure the system.
  6. When the installation is complete, reboot and remove the ISO if prompted.
  7. Log in to the Parrot desktop using the credentials you created.
6 Hyper-V Networking for Parrot

Verify Network Connectivity in Parrot

  1. Ensure your Parrot VM is connected to the External-Parrot-HTB virtual switch.
  2. Start Parrot and open a terminal.
  3. Check IP addressing with:
    ip a
  4. Confirm that the primary interface (e.g., eth0 or ens33) has an IP address.
  5. Test internet connectivity:
    ping -c 4 8.8.8.8

External Virtual Switch Behavior

Because you are using an external virtual switch, your Parrot VM appears on the same network as your host. This is convenient for platforms like Hack The Box that rely on VPN tunnels for lab connectivity.

7 Connecting Parrot OS to Hack The Box

Hack The Box (HTB) uses VPN tunnels to place your machine inside its lab networks. In this section, you will install OpenVPN (or the HTB VPN client, if specified), download your connection pack, and verify connectivity.

Step 1 – Update Package Lists

  1. In Parrot, open a terminal.
  2. Run:
    sudo parrot-upgrade
    or, if instructed:
    sudo apt update && sudo apt upgrade
  3. Allow updates to complete.

Step 2 – Install OpenVPN (if using OpenVPN pack)

  1. In the terminal, run:
    sudo apt install openvpn -y
  2. Verify there are no errors during installation.

Step 3 – Download Your HTB VPN Pack

  1. Open a browser inside Parrot and navigate to https://www.hackthebox.com.
  2. Sign in with your Hack The Box account.
  3. Locate the area where you download your VPN connection pack (often under “Connect” or “Access”).
  4. Select the appropriate VPN server/location and download the .ovpn configuration file.
  5. Save it to your Parrot home directory (e.g., ~/Downloads).
8 Step 4 – Connect Using OpenVPN
  1. Open a terminal in Parrot.
  2. Change to the directory where you saved the .ovpn file, for example:
    cd ~/Downloads
  3. List files to confirm the configuration file is present:
    ls
  4. Start the VPN tunnel (replace yourfile.ovpn with the real file name):
    sudo openvpn yourfile.ovpn
  5. Enter your sudo password if prompted.
  6. Watch the terminal output. When you see “Initialization Sequence Completed”, the VPN is active.
  7. Do not close this terminal while you need HTB access.

Step 5 – Verify HTB Connectivity

  1. Open a new terminal in Parrot (leave the OpenVPN terminal running).
  2. Confirm the presence of the VPN interface:
    ip a
  3. You should see an interface such as tun0 with an IP address in the HTB-assigned range.
  4. From your HTB dashboard, start a target machine and note its IP address.
  5. Ping the target from Parrot:
    ping -c 4 <HTB_target_IP>
  6. If you receive replies, your Parrot VM is successfully connected to Hack The Box.

  • Downloaded and installed Parrot OS in a Hyper-V virtual machine.
  • Configured an external virtual switch and verified Parrot’s network connectivity.
  • Used Parrot to establish a VPN connection to Hack The Box.
