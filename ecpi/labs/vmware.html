3 Connect Claude to the Docker Tool

‚è±Ô∏è 10-15 minutes

üîß Configuration Goal

We'll tell Claude Desktop: "When I say 'use the SQLite tools', run this Docker container and talk to it via MCP."

üìê The Configuration Structure

The official docs show a Claude config entry that uses Docker like this:

{
  "mcpServers": {
    "sqlite": {
      "command": "docker",
      "args": [
        "run",
        "--rm",
        "-i",
        "-v",
        "mcp-test:/mcp",
        "mcp/sqlite",
        "--db-path",
        "/mcp/test.db"
      ]
    }
  }
}

üîç Understanding the Configuration

  • docker run --rm -i ‚Äì Run container interactively, clean up when done
  • -v mcp-test:/mcp ‚Äì Use a Docker volume named mcp-test at /mcp in the container (persists DB file)
  • mcp/sqlite ‚Äì Use the SQLite MCP server image we just pulled
  • --db-path /mcp/test.db ‚Äì Store the SQLite database in that volume
3.1 Find the Claude Config File (Windows)

On Windows, the config file is typically at:

%APPDATA%\Claude\claude_desktop_config.json

üìù Steps to Open the Config File:

  1. Press Win + R to open the Run dialog
  2. Type: %APPDATA%\Claude
  3. Press Enter
  4. In the folder that opens, find claude_desktop_config.json
  5. Right-click it ‚Üí Open with ‚Üí Notepad (or your preferred editor)

‚ö†Ô∏è Can't Find the File?

If the file doesn't exist, you can create it:

  1. In the %APPDATA%\Claude folder
  2. Right-click ‚Üí New ‚Üí Text Document
  3. Name it exactly: claude_desktop_config.json
  4. Make sure the extension is .json, not .txt
3.2 Add the SQLite MCP Server Config

There are two likely scenarios:

üìÑ Scenario A: File is Empty or Almost Empty

Replace the entire contents with:

{
  "mcpServers": {
    "sqlite": {
      "command": "docker",
      "args": [
        "run",
        "--rm",
        "-i",
        "-v",
        "mcp-sqlite:/mcp",
        "mcp/sqlite",
        "--db-path",
        "/mcp/test.db"
      ]
    }
  }
}

Note: You can use mcp-sqlite or mcp-test as the volume name; just keep it consistent.

üìÑ Scenario B: File Already Has Other MCP Servers

Add a comma after the last server entry, then add the SQLite config:

{
  "mcpServers": {
    "existing-server": {
      "command": "...",
      "args": ["..."]
    },
    "sqlite": {
      "command": "docker",
      "args": [
        "run",
        "--rm",
        "-i",
        "-v",
        "mcp-sqlite:/mcp",
        "mcp/sqlite",
        "--db-path",
        "/mcp/test.db"
      ]
    }
  }
}

üíæ Save and Restart

  1. Save the config file (Ctrl+S in Notepad)
  2. Close Claude Desktop completely
  3. Restart Claude Desktop
  4. Wait a few seconds for it to connect to the MCP server
