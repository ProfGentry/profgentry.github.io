VMware Virtual Machine Laboratory

A Hands-On Introduction to Virtualization for Cybersecurity Students

Prepared for Students by Professor Jennifer Gentry
Estimated Time: 90–120 Minutes Technical Level: Beginner Windows VMware Required
1 Introduction

Virtual machines provide a safe, isolated environment for cybersecurity experimentation. In this lab, you will install VMware Workstation Player, deploy a Kali Linux virtual machine, configure its networking, and connect it to TryHackMe.

Learning Outcomes

  • Explain virtualization and its value in cybersecurity practice.
  • Install VMware Workstation and set up a Kali Linux virtual machine.
  • Understand NAT, Bridged, and Host-Only network modes.
  • Connect Kali to TryHackMe using OpenVPN.
2 VMware & Virtualization Concepts

What Is Virtualization?

Virtualization creates a simulated computing environment where multiple operating systems can run simultaneously on the same hardware without interfering with one another.

Why Cybersecurity Uses Virtual Machines

  • Isolate tools and malware safely.
  • Test configurations without risking the host.
  • Practice offensive and defensive techniques in controlled environments.

Why Use Kali Linux?

Kali provides a full suite of penetration testing tools and is widely adopted in security training. It is the recommended OS for TryHackMe exercises.

3 Install VMware Workstation Player

Follow these steps to download and install VMware Workstation Player.

Download VMware

Visit https://www.vmware.com and download VMware Workstation Player for Windows.

Install VMware

Run the installer and accept the default configuration settings.

4 Create & Install Kali Linux VM

This section walks you through creating and configuring your Kali Linux virtual machine.

Download Kali ISO

Access the installer ISO at: Kali Official Downloads

Create Virtual Machine in VMware

  • Select “Create a New Virtual Machine.”
  • Choose the Kali ISO file.
  • Name the VM “Kali for TryHackMe.”
  • Allocate 4 GB RAM and 2 CPU cores.

Complete Kali Installation

Follow the graphical installer, create your user account, and reboot into the Kali desktop.

5 Networking Modes

NAT (Recommended)

Offers internet access and isolation; ideal for TryHackMe.

Bridged

Places the VM directly on the local network. Useful for certain network simulations.

Host-Only

Isolated from the internet; ideal for malware or offline analysis.

6 Connecting to TryHackMe

This section covers installing OpenVPN, downloading your TryHackMe key, and verifying connectivity.

Install OpenVPN

sudo apt install openvpn -y

Connect to TryHackMe

sudo openvpn yourfile.ovpn
