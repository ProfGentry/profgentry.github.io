Virtualization Operating Systems

Overview

These labs introduce you to virtual machine platforms commonly used in academic and professional cybersecurity environments. You will work with both VMware and Hyper-V to create and manage guest operating systems safely on a single physical host.

Completing both labs will help you compare features, workflows, and use cases across different hypervisors, supporting more flexible and informed tool choices in future courses and projects.

Recommended Preparation

  • Basic familiarity with Windows desktop navigation.
  • Awareness of system resources (CPU, memory, and disk space).
  • Interest in operating systems, networking, or cybersecurity practice.
